Lesson 1 of codefirstgirls: HTML

Tags and what they mean:

We learned how to code a website like this using html tags

  1. <!DOCTYPE html> to start the page
  2. End with </html>
  3. Use <head> and </head> next, for the header that will show on the internet tab
  4. Then open your <body>
    • Here you can put headings with <h1> to <h6> and close each at the end
    • Or paragraphs with <p> and </p>
    • Even go crazy with some ordered (<ol>) and unordered (<ul>) lists like these. Don't forget to close them!
  5. Now close your </body>!
  6. Images are complicated...copy and paste ;)
